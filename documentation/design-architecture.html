Project: United States College Information System
Internal Release Number: 1.0

Components

The UIC has two components: handling data and visualization. These 2 components includes eight modules, together they realize the visualization of the dataset of the universities in the United States. The core modules handling data visualization includes the following three: information module, comparison module, and map module. Other modules are includes Visualization Base Module, Data Cleaning Module, Data Selection Module.
The components of this system are listed below by type:
  • Data Handler Component
    • C-00: Data Handler
  • Visualization Component
    • C-01: Visualization

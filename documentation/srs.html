Release Information

Project: USCIS

Introduction

The USCI system is designed to visualize multiple data that is collected by the IPEDS (Integrated Postsecondary Education Data System) in order to provide the users with a better understanding of over forty thousand universities, colleges and post-secondary educational schools in the United States.

 

Summery Functional Requirements

The core fuctions of our project are the visualizatios, including simple chart visualizations and a map visualization.

Other fuctions that may be implemented include user login system.

Please follow the link to the detailed list of Functions Required.

Non-Functional Requirements

Our performance requirement for the project is to load the web page in a reasonable amount of time. Most of the interactive functions should display results within one second to ensure the performance of the system. We are using D3, a JavaScript based language which loads all the data into the system at once. So our project cannot load a data set piece by piece, by rows or columns. This requires us to clean up the data set and reduce the size of the data set to ensure fast response time.
